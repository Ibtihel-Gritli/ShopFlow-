# ShopFlow 
# Système de Gestion d'une Boutique en Ligne

Application full-stack de gestion de boutique en ligne

## Backend (Spring Boot 3)
- Spring MVC (architecture REST)
- Spring Data JPA (persistance)
- Lombok (réduction du code boilerplate)
- Base de données : H2

## Frontend (Next.js)
- Framework React pour le rendu côté serveur
- Interface utilisateur responsive
- Communication avec l’API REST du backend

## Fonctionnalités principales
- Gestion des produits, catégories, stocks
- Panier d’achat et commandes
- Authentification des utilisateurs
- Dashboard administrateur / vendeur
